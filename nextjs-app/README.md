# Next.js Boilerplate

Minimal Next.js App Router starter designed to be reused across projects.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Branding

Set your app name in `.env.local`:

```bash
NEXT_PUBLIC_PLATFORM_NAME=My Product Name
```

## Scripts

- `npm run dev` - start development server
- `npm run build` - production build
- `npm run start` - run production server
- `npm run lint` - lint app
